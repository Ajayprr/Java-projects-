Operating Systems Scheduler Lab

COMPILE
In terminal, navigate to the directory that contains Scheduler.java, Process.java, and the input files. Once you are in the correct directory, run the following command to compile the program:
javac Scheduler.java Process.java

RUN
Run the following command where “inputfile.txt” is the desired input file (ensure that the input file is within the same directory as the source code):
java Scheduler input1.txt

RUN WITH VERBOSE FLAG
Run the following command where “inputfile.txt” is the desired input file (ensure that the input file is within the same directory as the source code):
java Scheduler --verbose input1.txt

EXAMPLE

RUN
javac Scheduler.java Process.java input1.txt
java Scheduler input1.txt

RUN WITH VERBOSE FLAG
javac Scheduler.java Process.java input1.txt
java Scheduler --verbose input1.txt
