Operating Systems Linker Lab
 
Provided with an input file as the first command line argument the program will print its output to the console

COMPILE 
In terminal, navigate to the directory that contains the linker package 
and run the following command:
javac linker/Linker.java


RUN
Run the following command where "input/file" is
the path to the input file (ensure that input file is within the same directory as the source code):
java linker.Linker input/file

EXAMPLE
javac linker/Linker.java
java linker.Linker linker/input3.txt

the commands above run the program on the input file input3.txt
